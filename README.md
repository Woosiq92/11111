# 계연주 11,111일 기념 웹사이트 🎉

계연주님(1995년 7월 4일생)의 11,111일 D-day를 축하하는 웹사이트입니다.

## 기능

- 계연주님의 생년월일(1995.07.04)로부터 11,111일 날짜 자동 계산
- 실시간 D-day 카운트다운 표시 (초 단위까지)
- 11,111일 당일 특별 축하 메시지 표시
- 배경음악 자동 재생 (KOTA the Friend - PNW)
- 방명록 기능 (이름, 메시지 작성 및 삭제)
- 귀여운 핑크 톤 디자인

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 자동으로 11,111일 D-day가 계산되어 표시됩니다
3. 실시간으로 남은 일수가 업데이트됩니다

## 파일 구조

- `index.html` - 메인 HTML 파일
- `style.css` - 스타일시트
- `script.js` - 날짜 계산, 방명록, 음악 제어 로직
- `KOTA the Friend - PNW.mp3` - 배경음악 파일
- `README.md` - 프로젝트 설명

## 기술 스택

- HTML5
- CSS3 (그라디언트, 애니메이션)
- JavaScript (Vanilla JS)

## 브라우저 호환성

모든 최신 브라우저에서 동작합니다.
